About Us


The Canteen Management System has been developed to override the problems prevailing in the practicing manual system. This software is supported to eliminate and in some cases reduce the hardships faced by this existing system. So, the purpose of Canteen Management System is to automate it, so that their valuable data/ information can be stored for a longer period with easy accessing and manipulation of the same. The main objective of this system is to manage the details of canteen like food items, bills, etc.

Canteen Management System is used to place the food orders and has two parts to it, namely the Customer side and Vendor side. Admin can select the items for the day and also perform functionalities like adding user, removing user, adding food items , removing food items, also look at the current orders block wise. This system can be used by customers to order items quicky. Also the order details are stored in the database.

This application avoids errors while entering the data, and also provides error message while entering invalid input. No formal knowledge is needed for the user to use this system. It can lead to error free, secure, reliable and fast management system. Thus, it is user-friendly.


The functionalities provided by Canteen Management System are:

    1. The customer can view all food items.
    2. The customer can enter the quantity of the food item selected, and order date will be generated automatically.
    3. The order can be placed on adequate Wallet balance.
    4. The order history will display the list of orders placed by a customer.
    5. The food Vendor can view the orders and accept/cancel it with proper comments.
